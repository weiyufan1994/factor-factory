### Step 5: Skewness 方向性 — abs() vs raw vs negated

发现 abs(skew) 优于 raw skew：
- abs(skew) 只关心肥尾幅度，不区分 crash/bubble
- raw skew 在 bubble 时压制信号，损失了部分有效暴露
- negated skew 的逻辑（bubble时放大）在A股方向错误

### Step 6: Kurtosis + Skewness 联合 Regime（最终推荐）

**PM 提出的关键洞察**：kurtosis 判断肥尾效应，skewness 判断左边尾部方向，无需 abs(skew)。

**kurt - skew 复合 regime**：
```
regime_crash = 1 + 0.5 * (kurtosis_zs - skewness_zs)

signal_kurt_skew = folded_zs * regime_crash
```

其中：
- kurtosis 高 → 两边肥尾都存在
- skewness 负（crash）→ 左边尾部更极端时，kurt-skew 值更大 → 信号放大
- skewness 正（bubble）→ kurt-skew 抵消部分放大效果

## Regime 指标定义

| 指标 | 计算方式 | 含义 |
|------|---------|------|
| skewness | rolling(20d return, 60d).skew() | 负偏=crash，正偏=bubble |
| kurtosis | rolling(20d return, 60d).kurt() | 肥尾程度（双向极端事件频率） |
| kurt - skew | kurtosis_zs - skewness_zs | crash regime 强度 |

## 机制解释

1. **folded 的逻辑**：在放量时，`|signal|` 代表异动幅度；`|signal|` 大 → 做空 → 在 crash regime 时更有效

2. **kurtosis 的角色**：识别"两边都在极端"的状态——这种状态预示着均值回归失效

3. **skewness 的角色**：识别方向——左偏（crash）时左边极端事件更多，加强信号；右偏（bubble）时信号温和

4. **kurt-skew 组合为何有效**：两者各司其职，kurtosis 提供存在性，skewness 提供方向性，形成完整 crash regime 判断

## 完整公式（最终推荐版）

**因子公式**：
```
alpha = ((adv60 < volume) ?
  (-1 * ts_rank(abs(delta(close, 7)), 60)) * sign(delta(close, 7)) : 0)

folded_alpha = -abs(alpha)
```

**Regime 权重**：
```
regime = 1 + 0.5 * (kurtosis_20d_zs - skewness_20d_zs)

final_signal = folded_alpha * regime
```

**简化表达**（等价）：
```
# 数学上 folded_alpha ≈ ((adv60 < volume) ? -ts_rank(abs(delta(close,7)),60) : 0)
# 所以最终信号 ≈ ((adv60 < volume) ? -ts_rank(abs(delta(close,7)),60) : 0) * regime
```
